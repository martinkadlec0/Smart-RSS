Import

The SMART export exports everyhing including IDs of folders and feeds which are numbers from 1 to n. Because of this, SMART import from another instance of Smart RSS might cause id collisions. You are advised to clear all data before using the SMART import!.

OPML export exports only list of feeds and it should be safe to use OPML import at any time.
